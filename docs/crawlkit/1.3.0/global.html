genericAnchors(delayopt)

A method that extracts all href's from any anchor on the page. Can be used as the return value of Finder#getRunnable. The delay can be set via {@CrawlKit#setFinder}.
Parameters:
Name Type Attributes Default Description
delay integer <optional>
 0 The delay after which the function searches for anchors.
